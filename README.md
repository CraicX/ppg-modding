# ppg-modding
People Playground Modding

This repo contains the source to mods I have contributed to.  Please Feel free to use them as a starting point in your own mods.

A Joplin Notebook is available with a lot of references from downloaded mods, source classes, etc...  Find it in the Joplin folder.  
Download Joplin here:
https://joplinapp.org/

Official People Playground Modding Wiki is here:
https://www.studiominus.nl/ppg-modding/

Official People Playground Snippets Repo:
https://github.com/mestiez/ppg-snippets
